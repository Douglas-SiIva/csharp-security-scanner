# 🔐 C# Security Scanner

Projeto em C# focado em aprendizado de Programação Orientada a Objetos (POO) e conceitos básicos de segurança.

## 🚀 Objetivo

Construir um scanner simples capaz de identificar vulnerabilidades comuns como SQL Injection e XSS.

## 🧠 Tecnologias

- C# (.NET 9.0)
- .NET

## 🏃‍♂️ Como executar

```bash
dotnet run
```

## 📚 Funcionalidades planejadas

- [ ] Scanner de SQL Injection
- [ ] Detector de XSS
- [ ] Relatórios de vulnerabilidades
- [ ] Interface de linha de comando
